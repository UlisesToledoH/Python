# Gestor de Tareas CLI

Una pequeña aplicación de consola en Python para gestionar tareas (CRUD) sobre un archivo JSON.

## Características

- Añadir tareas  
- Listar tareas pendientes/completadas  
- Marcar tareas como completadas  
- Eliminar tareas  

## Instalación

   cd gestor-tareas
